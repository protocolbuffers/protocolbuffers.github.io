    Class LazyField

    • java.lang.Object
      • com.google.protobuf.LazyFieldLite
        • com.google.protobuf.LazyField

    • public class LazyField
extends LazyFieldLite
      LazyField encapsulates the logic of lazily parsing message fields. It stores the message in a ByteString initially and then parse it on-demand. Most of key methods are implemented in LazyFieldLite but this class can contain default instance of the message to provide hashCode(), equals() and toString().
